# AI Multilingual Translator

A serverless, private, and secure translation application that runs entirely client-side in the web browser using Transformers.js.

## Features
- **Client-Side AI Translation**: Powered by `Xenova/nllb-200-distilled-600m` running in the browser.
- **Client-Side OCR**: Extract text from images using `Tesseract.js`.
- **Document Text Extractor**: Upload and parse TXT, PDF, and DOCX files.
- **Privacy First**: No server backend. All data stays in your browser.
